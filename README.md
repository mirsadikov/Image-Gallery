# WebTech.CW2.12860

## About

This is a simple images gallery application. It stores single image, title, time and description to database.

<br />

**Functionality**

- CREATE
- READ
- UPDATE
- DELETE

<br />

## Installation

<br />

1. Clone the Source Code

```bash
git clone https://github.com/wiut00012860/WebTech.CW2.12860
```

2. Install dependencies

```bash
npm install
```

3. Run application

```bash
npm start
```

<br />

## Routes and API

```shell
|- /
|- /create
|- /allimages
|- /allimages/:id
|- /allimages/:id/delete
|- /about
|- /api/v1/allimages
|- /api/v1/allimages/:id
```
